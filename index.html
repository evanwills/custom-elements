<option-builder> Custom element

What does it do?

<option-builder> allows less technical users to create a tilda/carat separated string (e.g. apple^Apple~banana^Banana~orange^Orange~pear^Pear~plum^Plum) used by ACU Conference Form Builder to build <SELECT> lists and radio/checkbox inputs for user input fields or merchandise options

The tilda/carat format is not very intuitive for non-technical users and not easy to get right for anyone. <option-builder> makes it possible for anyone to create options in a fairly intuitive way.
